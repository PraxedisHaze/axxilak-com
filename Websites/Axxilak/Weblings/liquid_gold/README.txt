THANK YOU FOR PURCHASING THE "LIQUID GOLD" TEMPLATE.
(Created by Axxilak.com)

This is a standalone HTML5 template. It does not require React, Vue, or WordPress. It runs on pure, high-performance code.

QUICK START GUIDE:

Unzip this folder.

Double-click index.html to preview the site in your browser.

To edit the text, right-click index.html and open it with any text editor (Notepad, TextEdit, VS Code).

HOW TO CUSTOMIZE:

To change the Name/Logo: The template currently says "AXXILAK" as a placeholder. Search for "AXXILAK" in the code and replace it with YOUR OWN business name.

To change the colors: Look for the :root section at the top of the CSS style block. Change --primary: #d4af37; to any hex color you like.

To change the email address: Scroll to the bottom <form> section and update the text.
